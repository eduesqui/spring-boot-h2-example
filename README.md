## Descripción
H2 es una base de datos temporal usada para agilizar el proceso de desarrollo. En este ejemplo
se muestra su facilidad de uso.

## Requisitos
* Maven
* Java
* Spring boot
* H2
## Instalación
Posicionarse en la carpeta raiz y ejecutar `mvn clean install`
